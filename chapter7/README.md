# WST 157 · Chapter 7: Interface Design

Public classroom presentation website for WST 157 Website Planning & Production.

## Open

Open `index.html` in a browser.

## Controls

- Next / Back buttons
- Left / right arrow keys
- Overview button
- Present button or `F` key

## Files

- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`

## Chapter focus

Chapter 7 covers interface design: wayfinding, navigation systems, links and controls, form usability, interactive states, and mobile interface patterns that help users act with confidence.
