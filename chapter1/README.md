# WST 157 · Chapter 1: Strategy & Website Planning

Static interactive presentation site for WST 157 Website Planning & Production.

## Open

Open `index.html` in any modern browser.

## Controls

- Next / Back buttons move through the deck.
- Arrow keys also move forward and backward.
- `F` toggles Present mode.
- Overview opens a slide map.

## Class sequence

1. Opening question
2. Strategy is not decoration
3. Diagnose · Guide · Act
4. Website purpose activity
5. Client wants vs. user needs
6. First-version priority picker
7. Scope slider
8. Site type planning questions
9. Roles and responsibilities
10. Website production road map
11. Project charter sections
12. Project charter example
13. Charter builder
14. Quick review
15. Exit ticket

Source material: Web Style Guide, Chapter 1: Strategy, by Patrick J. Lynch and Sarah Horton. Course sequence aligned with WST 157 Website Planning & Production syllabus.
