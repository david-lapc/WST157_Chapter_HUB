# WST 157 · Chapter 5: Site Structure

A public classroom presentation website for WST 157 Website Planning and Production.

## Open

Open `index.html` in a browser.

## Controls

- Next / Back buttons
- Arrow keys
- Spacebar for next slide
- Overview button for slide list
- Present button or `F` key for present mode

## Files

- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`

## Chapter focus

Chapter 5 covers site structure: semantic HTML, document structure, CSS and responsive styling, file structure, content management systems, publishing workflow, search engine visibility, friendly URLs, metadata, and site maps.
